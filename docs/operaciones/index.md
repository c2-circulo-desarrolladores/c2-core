# Operaciones

Esta sección contiene todo lo relacionado al flujo del producto a entregar (librerías).

Código -> Colaboración -> Mantenimiento

## Resumen

### Código

- Para **escribir código**, seguimos esta [guía de estilo Python](codigo/guia_estilo_python.md)
- Para **crear librerías**, seguimos esta [guía de estilo de librerías](codigo/guia_estilo_librerias.md)
- Para **gestionar dependencias**, usamos [uv](codigo/uv.md)
- Para **configurar el entorno**, usamos estas [extensiones](codigo/extensiones.md)

### Colaboración

- Para **trabajar con Git**, seguimos este [flujo de trabajo](colaboracion/git_workflow.md)
- (Si **aún no tienes git**, sigue esta [configuración](colaboracion/git_config.md))
- Para **usar GitHub**, seguimos esta [guía](colaboracion/github.md)

### Mantenimiento
- Para **usar GitHub Actions**, seguimos esta [guía](mantenimiento/github_actions.md)
- Para **versionar**, seguimos esta [guía](mantenimiento/versionamiento.md)
- Para **hacer lanzamientos descriptivos**, utilizamos un [release drafter](mantenimiento/release_drafter.md)
